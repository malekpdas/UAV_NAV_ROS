# extra_nodes

This package contains miscellaneous utility nodes for logging, analysis, and visualization.

## Nodes

- [odom_logger_node](descriptor/odom_logger_node.yaml): Logs odometry data to a CSV file.
- [topic_analysis_node](descriptor/topic_analysis_node.yaml): Compares GPS and Odometry data and generates summary plots.
- [plot_orientation](descriptor/plot_orientation.yaml): Real-time 3D visualization of vehicle orientation from odometry.

## Note

These nodes are primarily used for debugging and performance verification.
